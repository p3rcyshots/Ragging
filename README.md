# Ragging
This project aims at implementing a local RAG solution. This allows companies and users to have their files in one place and take advantage of local AI LLM models to process and retrive information from them as per requirement. You do not need online AI solutions for this.

Step 1
Use source_tuning.py script to create Vector Db of the source files with docx and xlsx formats. You can put them in a directory and give the path with -d argument.

Step 2
Initiate with ragging.py script. 

Read the documentation for complete information.
